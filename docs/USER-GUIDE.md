# User Guide

## Getting Started

Welcome to Icon Creator! This guide will help you create beautiful icons and vector graphics using our intuitive visual editor.

### First Launch

When you first open Icon Creator, you'll see:

1. **Canvas**: The main work area where you create your designs
2. **Toolbar**: Tools for creating shapes and selecting objects
3. **Layer Panel**: Shows all objects in your design
4. **Property Panel**: Edit properties of selected objects
5. **Right Sidebar**: Access to grids, guides, and settings

### Creating Your First Icon

Let's create a simple icon step by step:

1. **Select the Rectangle Tool**
   - Click the rectangle icon in the toolbar
   - Or press 'R' on your keyboard

2. **Draw a Rectangle**
   - Click and drag on the canvas to create a rectangle
   - Hold Shift while dragging to create a perfect square

3. **Customize the Shape**
   - With the rectangle selected, use the Property Panel to:
     - Change the fill color
     - Adjust the border color and width
     - Set the border radius for rounded corners

4. **Add More Shapes**
   - Try adding a circle using the Circle Tool (O)
   - Add text using the Text Tool (T)

5. **Organize with Layers**
   - Use the Layer Panel to reorder objects
   - Group related objects together
   - Toggle visibility with the eye icon

## Tools Overview

### Selection Tool (V)

The Selection Tool is used to select, move, and resize objects.

**Basic Selection:**

- Click on an object to select it
- Click and drag to create a selection box
- Hold Shift to add objects to your selection
- Hold Ctrl/Cmd to toggle object selection

**Moving Objects:**

- Drag selected objects to move them
- Use arrow keys for precise movement (1px increments)
- Hold Shift + arrow keys for larger movement (10px increments)

**Resizing Objects:**

- Drag the corner handles to resize
- Hold Shift while resizing to maintain proportions
- Hold Alt while resizing to resize from center

**Rotating Objects:**

- Drag the rotation handle (circular arrow)
- Hold Shift while rotating to snap to 15-degree increments

### Rectangle Tool (R)

Create rectangular shapes and perfect squares.

**Creating Rectangles:**

- Click and drag to create a rectangle
- Hold Shift to create a perfect square
- The rectangle will be created with default styling

**Rectangle Properties:**

- **Fill Color**: The interior color of the rectangle
- **Border Color**: The outline color
- **Border Width**: Thickness of the outline
- **Border Radius**: Rounds the corners of the rectangle

### Circle Tool (O)

Create circles and ellipses.

**Creating Circles:**

- Click and drag to create an ellipse
- Hold Shift to create a perfect circle
- Start from where you want the center to be

**Circle Properties:**

- **Fill Color**: Interior color
- **Border Color**: Outline color
- **Border Width**: Outline thickness

### Text Tool (T)

Add text to your designs.

**Creating Text:**

- Click where you want to place text
- Type your text content
- Press Escape or click elsewhere when finished

**Text Properties:**

- **Font Family**: Choose from available fonts
- **Font Size**: Adjust text size
- **Font Weight**: Bold, normal, light options
- **Color**: Text color
- **Alignment**: Left, center, right alignment

**Editing Text:**

- Double-click text objects to edit content
- Use the Property Panel to change styling

### Path Tool (P)

Create custom shapes with curved and straight lines.

**Creating Paths:**

- Click to place path points
- Drag while clicking to create curved points
- Double-click to finish the path

**Path Editing:**

- Select a path to see its control points
- Drag points to reshape the path
- Add new points by clicking on path segments

## Working with Layers

### Layer Panel

The Layer Panel shows all objects in your design in a hierarchical list.

**Layer Operations:**

- **Reorder**: Drag layers up or down to change stacking order
- **Group**: Select multiple objects and right-click to group
- **Visibility**: Click the eye icon to show/hide objects
- **Lock**: Click the lock icon to prevent editing

**Layer Names:**

- Double-click a layer name to rename it
- Use descriptive names for better organization

### Grouping Objects

Groups help organize related objects:

1. Select multiple objects (hold Shift while clicking)
2. Right-click and choose "Group" or press Ctrl/Cmd + G
3. Groups can be nested inside other groups
4. Ungroup with Ctrl/Cmd + Shift + G

## Property Editing

### Common Properties

Most objects share these common properties:

**Position:**

- **X**: Horizontal position from left edge
- **Y**: Vertical position from top edge

**Size:**

- **Width**: Object width
- **Height**: Object height
- **Lock Aspect Ratio**: Maintain proportions when resizing

**Appearance:**

- **Fill Color**: Interior color
- **Border Color**: Outline color
- **Border Width**: Outline thickness
- **Opacity**: Transparency level (0-100%)

**Transform:**

- **Rotation**: Rotate object in degrees
- **Scale X/Y**: Stretch or compress horizontally/vertically

### Precision Editing

Use the Precision Inputs for exact positioning:

1. Open the Precision Panel in the right sidebar
2. Enter exact values for position, size, and rotation
3. Use mathematical expressions (e.g., "100 + 50" = 150)

## Grid and Alignment

### Grid System

The grid helps align objects precisely:

**Grid Controls:**

- Toggle grid visibility with the Grid button or 'G' key
- Adjust grid spacing in the Grid Panel
- Change grid color and opacity

**Snap to Grid:**

- Enable snap-to-grid for automatic alignment
- Objects will snap to grid intersections when moved
- Toggle with Shift + G

### Alignment Tools

Align multiple objects relative to each other:

1. Select multiple objects
2. Use alignment buttons in the Property Panel:
   - **Align Left/Right**: Align to leftmost/rightmost object
   - **Align Top/Bottom**: Align to topmost/bottommost object
   - **Center**: Align to center horizontally or vertically

**Distribution:**

- **Distribute Horizontally**: Even spacing between objects horizontally
- **Distribute Vertically**: Even spacing between objects vertically

### Manual Guides

Create custom alignment guides:

1. Open the Guides Panel in the right sidebar
2. Click "Add Horizontal Guide" or "Add Vertical Guide"
3. Drag guides to reposition them
4. Objects will snap to guides when nearby

## Themes and Customization

### Theme Modes

Icon Creator supports multiple theme modes:

- **Light**: Traditional light interface
- **Dark**: Dark mode for low-light environments
- **Auto**: Automatically matches your system preference
- **Custom**: Create your own personalized theme

### Custom Themes

Create custom themes to match your workflow:

1. Open Theme Settings in the right sidebar
2. Choose "Custom" theme mode
3. Customize colors:
   - **Primary Color**: Main accent color
   - **Background Color**: Main background
   - **Text Color**: Primary text color
   - **Surface Color**: Panel backgrounds
   - **Border Color**: Panel borders

4. Adjust spacing:
   - **Base Unit**: Fundamental spacing unit
   - **Small/Medium/Large**: Predefined spacing sizes

5. Save your custom theme for reuse

### Importing/Exporting Themes

Share themes with others:

- **Export**: Save your custom theme as a JSON file
- **Import**: Load a theme file created by someone else

## Keyboard Shortcuts

### Essential Shortcuts

**Tools:**

- V: Selection Tool
- R: Rectangle Tool  
- O: Circle Tool
- T: Text Tool
- P: Path Tool

**Edit Operations:**

- Ctrl/Cmd + C: Copy
- Ctrl/Cmd + V: Paste
- Ctrl/Cmd + X: Cut
- Ctrl/Cmd + D: Duplicate
- Delete: Delete selected objects

**Selection:**

- Ctrl/Cmd + A: Select All
- Escape: Deselect All

**View:**

- G: Toggle Grid
- Shift + G: Toggle Snap to Grid
- Ctrl/Cmd + 0: Fit All in Window
- Ctrl/Cmd + 1: Zoom to 100%
- Ctrl/Cmd + Plus: Zoom In
- Ctrl/Cmd + Minus: Zoom Out

**History:**

- Ctrl/Cmd + Z: Undo
- Ctrl/Cmd + Y: Redo

### Advanced Shortcuts

**Object Manipulation:**

- Arrow Keys: Move 1px
- Shift + Arrow Keys: Move 10px
- Ctrl/Cmd + Arrow Keys: Move 0.1px (precision)

**Layer Operations:**

- Ctrl/Cmd + G: Group
- Ctrl/Cmd + Shift + G: Ungroup
- Ctrl/Cmd + ]: Bring Forward
- Ctrl/Cmd + [: Send Backward

## Import and Export

### Importing Files

Import existing SVG files to edit:

1. Click the Import button in the toolbar
2. Select an SVG file from your computer
3. The file will be imported as editable objects
4. Use the Layer Panel to see imported objects

**Supported Formats:**

- SVG files (.svg)
- Icon files with SVG content

### Exporting Your Work

Export your creations in multiple formats:

**SVG Export:**

- Vector format, scalable to any size
- Ideal for web use and further editing
- Preserves all object properties

**PNG Export:**

- Raster format for pixel-perfect images
- Choose resolution (72 DPI for web, 300 DPI for print)
- Select export size

**JSON Export:**

- Native format preserving full editing capability
- Import back into Icon Creator later
- Includes layer structure and all properties

### Export Options

**Quality Settings:**

- **Low**: Faster export, smaller files
- **Medium**: Balanced quality and file size
- **High**: Best quality, larger files

**Size Options:**

- **Original**: Export at current canvas size
- **Custom**: Specify exact pixel dimensions
- **Scale**: Export at multiple sizes (1x, 2x, 3x)

## Accessibility Features

### Keyboard Navigation

Icon Creator is fully accessible via keyboard:

- **Tab/Shift+Tab**: Navigate between interface elements
- **Enter/Space**: Activate buttons and controls
- **Arrow Keys**: Navigate within panels and menus
- **Escape**: Close dialogs and cancel operations

### Screen Reader Support

All interface elements include proper labels for screen readers:

- Object descriptions include type, position, and properties
- Live announcements for important state changes
- Descriptive labels for all tools and controls

### High Contrast Mode

Enable high contrast for better visibility:

1. Open Accessibility Settings
2. Toggle "High Contrast Mode"
3. Interface will use enhanced contrast ratios

### Reduced Motion

For users sensitive to animation:

1. Open Accessibility Settings
2. Enable "Reduce Motion"
3. Animations will be minimized or disabled

## Troubleshooting

### Performance Issues

**Large Files:**

- Use groups to organize complex designs
- Hide unnecessary layers while working
- Consider breaking very large designs into multiple files

**Slow Rendering:**

- Reduce the number of visible objects
- Lower the zoom level when working with many objects
- Close other applications if memory is limited

### Common Problems

**Objects Not Selectable:**

- Check if the object is locked (lock icon in Layer Panel)
- Ensure the object is visible (eye icon in Layer Panel)
- Try clicking directly on the object, not empty space

**Unexpected Behavior:**

- Try refreshing the page to reset the application
- Check if you're in the right tool mode
- Ensure you have objects selected for operations that require them

**Import Issues:**

- Verify the SVG file is valid and well-formed
- Check file size - very large files may take time to process
- Try simplifying the SVG in another editor first

### Getting Help

**Built-in Help:**

- Hover over tools and buttons for tooltips
- Check the onboarding tour if you're new
- Use the keyboard shortcut reference (press ? key)

**Reporting Bugs:**

- Use the feedback button in the application
- Include steps to reproduce the issue
- Mention your browser and version
